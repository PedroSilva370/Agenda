
# Agenda App

A simple command-line agenda application made with Python.

> **Note:** The source code is written in Portuguese.

---

## Features

- Add appointments
- View agenda
- Edit appointments
- Remove appointments
- Search by ID

---

## Technologies

- Python 3
- Standard library (`time`)

---

## Running the Project

```bash
python main.py
```

---

## Menu

```text
[ 1 ] Add appointment
[ 2 ] View agenda
[ 3 ] Edit appointment
[ 4 ] Remove appointment
[ 5 ] Search by ID
[ 0 ] Exit
```
